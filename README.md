# Git & GitHub Starter Challenge

### The Problem Statement
Practice the professional "Feature Branch" workflow by forking, cloning, and contributing to this repository without touching the `main` branch directly.

### Project Tasks
1. **Fork** this repo to your account.
2. **Clone** it locally to your machine.
3. **Branch**: Create a branch named after yourself
4. **Modify**: Update the `index.html` file with your details.
5. **Document**: List the commands you used in the "Command History" section below.
6. **PR**: Push your branch and open a Pull Request.

### Command History 
Complete this README.md with the commands you used to complete the tasks above.
